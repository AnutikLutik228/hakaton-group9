Email Sorter (Автоматическая сортировка писем)

КАК ПОЛЬЗОВАТЬСЯ: Положи свои письма (.eml или .txt) в папку inbox. Она обязательно должна находиться в корневой директории проекта, либо автоматический запуск будет невозможен.

---

### ДЛЯ macOS / Linux:

АВТОМАТИЧЕСКИЙ ЗАПУСК ЧЕРЕЗ СКРИПТ (рекомендуется): 
Убедись, что у тебя есть права на запуск (chmod +x run.sh), и выполни: 
`./run.sh`

ВРУЧНУЮ (если папка inbox в другом месте): 
Активируй окружение (source .venv/bin/activate) и запусти скрипт с нужным путем: 
`python3 scr/main.py --inbox путь/к/твоей/папке` 

Чтобы проверить работу без перемещения файлов, добавь флаг --dry-run: 
`python3 scr/main.py --inbox inbox --dry-run`

---

### ДЛЯ Windows:

АВТОМАТИЧЕСКИЙ ЗАПУСК ЧЕРЕЗ СКРИПТ (рекомендуется): 
Выполни файл запуска в командной строке или PowerShell: 
`.\run.bat`

ВРУЧНУЮ (если папка inbox в другом месте): 
Активируй окружение (.\.venv\Scripts\activate) и запусти скрипт с нужным путем: 
`python scr\main.py --inbox путь\к\твоей\папке` 

Чтобы проверить работу без перемещения файлов, добавь флаг --dry-run: 
`python scr\main.py --inbox inbox --dry-run`

---

РЕЗУЛЬТАТ: 
Рассортированные письма появятся в папках с названиями категорий. Неопределенные письма попадут в папку broken. Статистика будет сохранена в report.txt.
